# Melbourne Housing Regression

This project uses housing data from Domain.com.au posted on Kaggle, the project is focused on doing analysis on the data, as well as creating multiple models such as Linear Regression, Decision Trees, and Random Forest to predict the price of a home in Melbourne, Australia. This is done to gain a better insight into housing prices as well as gain a market advantage though the potential use of the targeted advertising with the help of the modeling.

# Files and Folders

* The melbourne-housing-market.html is the final output for the project, [it can be viewed here...](https://justingee193.github.io/justingee193.github.io/links/melbourne-housing-market.html)
* The melbourne-housing-market.rmd file is where the code and output is written
* The Melbourne_housing_FULL.csv file contains the data used for the project
